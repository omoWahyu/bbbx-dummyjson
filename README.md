# Introductions
this stack is use:
React, TypeScript, Vite and Bun

## Installation

1. Install Bun
``` bash
curl -fsSL https://bun.sh/install | bash
```
or in homebrew
``` bash
brew install oven-sh/bun/bun # for macOS and Linux
```
or in windows, using scoop
``` powershell
scoop install bun
```

2. Install the depedency
``` bash
bun install
```

3. Start the app (client side)
``` bash
bun run dev
```
